Wolford

Wolford is an Action RPG where the player can choose from 1 of 3 upgradable characters to explore the different levels killing enemies and completing the levels to prepare for the final boss fight. My contributions to this project include:

  • Level Design for the second level
  • Design and Implementation for the spawning system
  • Design and Implementation of the drop system
  • Design and Implementation of the Level Progression system
  • Implemented loading screen/system
  • Input indicators
  • Audio in different parts of the game
  • QA testing / Fixing bugs as they arise

  • Project Type: Action RPG
  • Date: 09/2021 - 10/2021
  • Roles: Game Developer Game Designer Level Designer
  • Tools: Unity Github Trello

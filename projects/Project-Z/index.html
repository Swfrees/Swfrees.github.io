Project-Z

Project-Z is a FPS Maze Runner Puzzle game where the player must complete each puzzle contained within the maze to complete the level all while surviving the endless hordes of Zombies hunting them down.
My contributions to this project include:

  • Level design for all 8 mazes
  • Design and implementation of the multiple puzzle systems found throughout the game
  • Design and implementation of the difficulty system used throughout the game
  • Design and implementation for the spawning system used to spawn in enemies

  • Project Type: Puzzle maze-runner FPS
  • Date: 07/2022 - 10/2022
  • Roles: Game Developer Game Designer Level Designer
  • Tools: Unreal Engine 4 Perforce Blender Trello

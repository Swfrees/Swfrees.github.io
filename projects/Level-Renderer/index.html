Level Renderer

This project was created using the Vulkan graphics API with assistance from the Gateware software from Gateware Development. The goal was to get multiple objects rendering within a 3D space over 1 or more levels

  • Project Type: Rendering Simulation
  • Date: 03/2022 - 04/2022
  • Roles: Software Engineer Level Designer
  • Tools: Vulkan Gateware Github Blender
